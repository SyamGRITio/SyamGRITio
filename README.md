## Philosophy

実装や構築の完了そのものだけでなく、その後の安定稼働、可視性、オブザーバビリティ、そして開発者が円滑にデバッグや監視を行える環境づくりを重視しています。

インフラの役割は、単に基盤を用意することではなく、プロジェクト全体が継続的に前進しやすい状態を支えることにあると考えています。  
そのため、インフラの理解者として、チームやプロジェクトの潤滑油となることを大切にしています。
